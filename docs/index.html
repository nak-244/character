    募集内容

    当社が運営する総合求人サイト【しごとら】のマスコットキャラクターのデザインを募集致します。

    【しごとら】は正社員・派遣・パート・アルバイトなど多様な働き方から仕事を探すことができ、給与の支払いスパンも「日払い」「週払い」「月払い」から選べるのが特徴です。

    おかげさまで、【しごとら】のユーザー数が昨年の2.7倍と増加傾向にあります。マスコットキャラクターを起用することにより、身近な「サービス認知」の拡大を目指すとともに、【しごとら】のサービスがどのようなものなのかを直感的に伝え、今後の広告宣伝をより効果的なものとしていくことを目的とします。

    公式ホームページをはじめ、SNS、各種グッズ、チラシ等、当社のさまざまな広告宣伝活動において、多種多様に展開することを前提としております。
